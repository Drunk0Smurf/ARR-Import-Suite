🍪 Cookie Manager
Session cookies for private trackers · auto-strips password fields on paste
How to get your session cookie
  1. Log into the tracker in your browser
  2. Press F12 → Network tab → refresh the page
  3. Click any request to that tracker's domain → Headers tab
  4. Find Cookie: in Request Headers — copy the entire value
  5. Copy the User-Agent: value from the same section
  6. Paste both below — password fields like oldpass are stripped automatically
  7. Click Save & Push to Prowlarr — if Prowlarr is connected it updates instantly
